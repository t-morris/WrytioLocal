# WrytioLocal
Client-side (Local) Version of Wrytio - a project to allow writers to simply track words written across multiple projects & provide tracking and trendlines for writing challenges (such as Nanowrimo).

Requires:
- jQuery (3.5.1)
- popper.js (1.16.0)
- Bootstrap (4.4).1)
- Flatly Theme (4.5.2)
- Darkly Theme (4.5.2)
- datejs (1.0)
- Chart.js (2.9.3)
